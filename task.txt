Сделать single page application на Vue.js, с бэкэндом на Python (3.6+) в котором на главном экране показывать список загруженных текстов и поле куда человек в браузере может вставить из буфера обмена любой небольшой текст на английском языке (до 1Mb).

При загрузке текст сохраняется и разбивается на предложения. В интерфейсе у каждого текста будет уникальная ссылка, которая показывает документ как список предложений.

После добавления текста пользователю показывается страница текста (список предложений).

При клике на предложение нужно находить другие тексты в которых есть похожие предложения и выводить предложения как список отсортированный по смысловой близости к выбраному предложению (от наиболее похожего к наименее похожему). В списке выводить само предложение, ссылку на текст содержащий это предложение, значение метрики похожести. Смысловую близость предложений считать word embedding (word2vec, fasttext) или любым другим более современным способом.

Код оформить так как обычно оформляют продакшен код и выложить как открытый репозиторий на GitHub или BitBucket и прислать ссылку.

Опционально, как бонус добавить в репозиторий docker-compose файл чтобы одной командой docker-compose up можно было поднять весь стек приложения.
